# CLAUDE.md

## Project: FileChest (django-filechest)

A simple-like file manager as a Django app.

## Language Policy

- All code (comments, docstrings, variable names) must be in English
- All documentation must be in English
- Git commit messages must be in English

## Commands

```bash
uv run python manage.py runserver   # Run dev server
uv run python manage.py test        # Run tests
uv run python manage.py migrate     # Apply migrations
```

## Project Structure

- `django_filechest/` - Django project settings (for development)
- `filechest/` - The reusable Django app (this becomes the PyPI package)
- `docs/` - Design documents

## Static Files

- Edit static files in `filechest/static/` only
- `django_filechest/staticroot/` is generated by `collectstatic` - do not edit directly

## Git Workflow

- Do not commit unless explicitly instructed by the user
